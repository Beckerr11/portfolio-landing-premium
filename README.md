# Portfolio Landing Premium

Landing publica premium para portfolio, com paginas separadas de inicio, servicos, projetos, sobre, curriculo e cases.

![Preview da landing](./docs/preview-landing-home.png)

## Paginas

- inicio
- servicos
- projetos
- sobre
- curriculo online
- case studies

## Stack

- React
- Vite
- React Router
- React Icons

## Como rodar

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run lint
npm run build
```

## Estrutura principal

- `src/pages/Portfolio.jsx`: home comercial
- `src/pages/Services.jsx`: catalogo de servicos
- `src/pages/Projects.jsx`: vitrine de projetos
- `src/pages/Curriculo.jsx`: curriculo online
- `src/pages/ProjectCaseStudy.jsx`: cases individuais

## Observacoes

- Esta base e apenas a vitrine publica.
- Rotas antigas do workspace redirecionam para a demo ao vivo.
- Preparada para deploy em Vercel como site estatico.
